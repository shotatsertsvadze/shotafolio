Project Overview

In this one, I incorporated the NASA API to get random astronomy pictures of the day. you can see that the results are formatted into cards with an image title.

If we click on an image, we can see that we can view the full resolution version of that image in a new tab.If we close that, we also have the ability to add images to our favorites page. This is accomplished using local storage.
